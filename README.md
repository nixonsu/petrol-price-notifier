# Petrol Price Notifier
Daily cron job that pulls and publishes lowest petrol prices for the day. 

## Technologies

- Kotlin
- Gradle
- AWS SDK
- AWS Lambda, EventBridge, SNS, Cloudformation
- GitHub Actions
